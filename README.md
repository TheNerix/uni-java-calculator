## JAVA CALCULATOR ##

A simple calculator built as a part of my university Java programming course.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Console-based menu
- Repeated calculations until the user exits the program

## Technologies

- Java
- Java Scanner


## About the project

This calculator was developed during my university Java programming course.

The project was a practice basic Java concepts such as:

- classes and objects
- methods
- variables and data types
- user input
- loops
- switch statements
- integer and floating-point arithmetic
